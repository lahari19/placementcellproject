## Placement Website
This is a placement website for IIITDM Kancheepuram.

## For developers (Initial Setup)
- clone this repo
- always change branch to dev :: git checkout dev
- nodemon,express installation required 
- run -> sudo npm i nodemon -g
- run -> sudo npm i
# To start server
- run -> npm run dev
 ### or 
- node app.js
